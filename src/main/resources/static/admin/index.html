한국사 학습 데이터 관리 시스템

체계적이고 효율적인 한국사 학습을 위한 7단계 계층적 데이터 관리 시스템입니다. 선사시대부터 현대까지, 모든 한국사 학습 자료를 체계적으로 관리하고 학습할 수 있습니다.

🚀 주요 기능

📚

계층적 데이터 구조

Chapter → Lesson → Section → Subsection → Topic → Keyword → Content의 7단계 계층적 구조로 체계적인 학습 자료 관리

🔍

강력한 검색 기능

키워드 기반 부분 검색으로 원하는 학습 자료를 빠르게 찾을 수 있습니다. 다중 키워드 검색도 지원합니다.

⚡

완전한 CRUD 지원

생성, 조회, 수정, 삭제의 모든 기능을 지원하며, Cascade 삭제로 데이터 일관성을 보장합니다.

🎯

직관적인 관리 인터페이스

사용자 친화적인 웹 인터페이스로 쉬운 데이터 관리가 가능합니다. 실시간 데이터 업데이트를 지원합니다.

🔗

RESTful API

표준 REST API를 제공하여 다른 시스템과의 연동이 가능합니다. JSON 형태의 데이터 교환을 지원합니다.

💾

안정적인 데이터 저장

H2 데이터베이스를 사용하여 빠르고 안정적인 데이터 저장을 제공합니다. 메모리 기반으로 빠른 성능을 보장합니다.
