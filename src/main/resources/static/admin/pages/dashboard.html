📊 시스템 개요

KoreaHistory는 한국사 학습을 위한 7단계 계층적 데이터 관리 시스템입니다.

1단계 Chapter (대분류) - 선사시대, 고대, 중세 등 시대별 대분류
2단계 Lesson (중분류) - 각 시대 내 세부 주제
3단계 Section (소분류) - 주제별 세부 분류
4단계 Subsection (상세분류) - 더욱 세분화된 주제
5단계 Topic (주제) - 구체적인 학습 주제
6단계 Keyword (키워드) - 핵심 키워드 그룹
7단계 Content (내용) - 상세 학습 내용

Keyword 검색 및 수정

🔗 API 엔드포인트

메서드 엔드포인트 설명 관리 페이지
GET /api/v1/chapters/search/all 모든 대분류 조회 Chapter 관리
POST /api/v1/create/chapter 대분류 생성 Chapter 관리
PATCH /api/v1/chapters/{id}/title 대분류 제목 수정 Chapter 관리
DELETE /api/v1/chapters/{id} 대분류 삭제 Chapter 관리
GET /api/v1/search/keywords 키워드 검색 Keyword 관리
POST /api/v1/create/keyword 키워드 생성 Keyword 관리
POST /api/v1/create/section/{lessonId} 소분류 생성 Section 관리
POST /api/v1/create/subsection/{sectionId} 상세분류 생성 Subsection 관리
